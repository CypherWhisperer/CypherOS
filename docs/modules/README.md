# CYPHER OS MODULES DOCUMENTATION SOURCE OF TRUTH

- This directory is the single source of truth for all modules (source files) within CypherOS.
- The directory tree structure is an identical mirror of the `modules/` directory from CypherOS's root.
- This gives me the flexibility to work on md files from Obsidian (Obsidian follows symlinks) whilst having those changes propagate to the IDE;

> **NOTE: Version control for the md files related to the repo are version controlled with the repository; Obsidian only acts as the editting layer**
